# 24 Hours Code Challenge – Web Shop Homepage

## Overview
This project is a front-end implementation of a web shop homepage, developed as part of a 24-hour code challenge. The goal was to create a **pixel-perfect, responsive homepage** based on a provided wireframe/mockup.  

The challenge was designed to assess:  
1. Approach to implementing a modern, responsive e-commerce front-end.  
2. Familiarity with current web development standards, including SCSS, HTML5, CSS3, and JavaScript.  
3. Use of SCSS and the Foundation framework for structuring a scalable front-end project.  

---

## Technologies Used
- **HTML5 & CSS3** – Semantic markup and styling  
- **SCSS / SASS** – Modular and maintainable styles  
- **Foundation Framework** – Responsive grid system and utility classes  
- **Flickity** – For carousels and product galleries  
- **Google Fonts** – For web typography  
- **JavaScript (vanilla)** – Interactions and carousel initialization  

---

## Installation

Execute the following commands

```bash
git clone https://github.com/Jordiak/D1-Site.git
cd d1-site
npm install
```

Finally, run the command `npm start` to initialize the site. The site will display at:

```
http://localhost:8000
```
